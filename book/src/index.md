metaBuilder describes builders as typed values. The same spec can
drive validation, schema extraction, documentation, dependency
graphs, dry runs, and materialization. Source and releases live on
[GitHub](https://github.com/kleisli-io/metaBuilder).

## Manual

Start with the [Manual](/metaBuilder/manual) to learn how to
write and use metaBuilder builders as typed descriptions.

## Theory

Read [Theory](/metaBuilder/theory) for the mathematical framing:
descriptions, internalized programs, ornaments, interpretations,
and runtime structure.

## Examples

The [Examples](/metaBuilder/examples) are guided artifact tours
with source pages and generated views.
